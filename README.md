**使用方法**

1.下载release里的zip压缩包，双击运行exe

此时主程序会进行初始化，创建各种文件夹之类的(其实就两个，一个在运行目录用来存放翻译文件，还有一个在AppData目录里用来保存游戏路径)

2.等程序初始化完成(会弹窗提醒)，将外部链接中下载到的翻译文件(translate.xlsx)放到运行目录下的translate文件夹

3.再次双击运行。此时若为第一次运行程序会自动开始扫描磁盘定位游戏，若需要查看进度可打开log.log文件查看(可能存在轻微滞后)。定位完成后会自动安装modloader和plugin，可能会花费几分钟，所以可能得耐心等一下awa
